# E-commerce Product Listing App

![App Logo](public/assets/logo.png)

A modern e-commerce product listing app built with **Next.js**, **TypeScript**, and **Tailwind CSS**. Features reusable components, clean folder structure, and scalable architecture for building a full shopping experience.

---

## Project Description
E-commerce Product Listing App is designed to display products in a clean, responsive listing page. The goal is to practice building a fully functional frontend with **Next.js** and **Tailwind CSS**, including reusable components, static assets, and TypeScript type safety. This app can serve as the foundation for an online store interface.

---

## Project Structure

- **components/** – Reusable React components such as `Card` for product display, buttons, and icons.
- **interfaces/** – TypeScript interfaces and types to define the structure of product data.
- **constants/** – Constant values used across the app, including sample product data.
- **public/assets/** – Static assets such as product images, placeholders, and SVG icons.

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**

```bash```
git clone https://github.com/Hopeson-Jockiel/ecommerce-product-listing.git

2. Navigate to the project directory

cd ecommerce-product-listing


3. Install dependencies

npm install


4. Start the development server

npm run dev


5. Open the app in your browser

Visit http://localhost:3000
 to see the app in action.
